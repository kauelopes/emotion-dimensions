# The Shape of Emotion

Interactive scrollytelling site for PhD research on the intrinsic dimensionality
of the emotion lexicon in pretrained embedding spaces.

Live: https://kauelopes.github.io/emotion-dimensions/

## Stack

Next.js (App Router, static export) · Tailwind CSS v4 · framer-motion ·
three.js via @react-three/fiber + drei.

## Develop

```bash
npm install
npm run dev      # http://localhost:3000
npm run build    # static export -> out/
```

## Data

`public/data/*.json` are generated by `src/export_site_data.py` in the
(private) research repository and committed here. To refresh them, run
`uv run python src/export_site_data.py` there and commit the result.

Deployment is automatic: every push to `main` builds and publishes to GitHub
Pages via `.github/workflows/deploy.yml`.
